# Polymarket 监控看板

实时追踪 Polymarket 预测市场数据的专业监控工具。

## 🔗 数据来源

所有数据均来自 Polymarket 官方 API：
- **排行榜数据**: `https://data-api.polymarket.com/v1/leaderboard`
- **持仓数据**: `https://data-api.polymarket.com/positions`
- **交易历史**: `https://data-api.polymarket.com/activity`
- **市场数据**: `https://gamma-api.polymarket.com/markets`

## 📊 功能特性

### 1️⃣ 总览面板 - 盈利钱包排行榜

**类别筛选**:
- 全部类别 / Politics 政治 / Sports 体育 / Crypto 加密
- Finance 金融 / Culture 文化 / Weather 天气
- Economics 经济 / Tech 科技

**时间筛选**:
- 今天 / 周度 / 月度 / 全部

**排行榜字段** (点击表头可排序):
| 字段 | 颜色 | 说明 |
|------|------|------|
| 排名 | 🟡 黄色 | 当前排名 |
| 钱包地址 | 🔵 蓝色 | 完整地址，点击跳转详情 |
| 盈利/亏损 | 🟢/🔴 绿/红 | 盈亏金额 |
| ROI | 🔷 青色 | 投资回报率 |
| 胜率% | 🟠 橙色 | 获胜百分比 |
| USDC余额 | 🟣 紫色 | 账户余额 |
| 持仓情况 | 🩷 粉色 | 持仓总价值 |
| 交易次数 | 🔵 靛蓝 | 总交易数 |
| 平均投注 | 🔵 蓝色 | 平均每笔投注 |
| 平均盈利 | 🟢 绿色 | 平均每笔盈利 |
| 投入资金 | 🟡 黄色 | 总投入金额 |
| 活跃天数 | ⚪ 灰色 | 活跃时间 |
| 日均交易 | 🔷 青色 | 每日平均交易 |
| 总成交量 | 🔵 蓝色 | 累计成交额 |
| 主要类别 | 彩色标签 | 中英文显示 |
| 最近交易 | ⚪ 灰色 | 相对时间 |

**分页**: 每页50名，共50页（最多2500名）

### 2️⃣ 聪明钱追踪面板

- 🔥 热门市场 (实时赔率、成交量、流动性)
- 🐋 实时大额交易 (买入/卖出信号)
- 交易金额筛选: $1K+ / $5K+ / $10K+ / $50K+

### 3️⃣ 账户详情面板

**第一行 - 主要指标**:
- USDC余额、盈利/亏损、ROI、Portfolio价值
- 平均投注、平均盈利、投入金额、未实现盈亏
- 总成交量、主要类别

**第二行 - 交易统计**:
- 交易次数、买入、卖出、买入额、卖出额
- 参与市场、活跃天数、ROI、胜率

**第三行 - 持仓统计**:
- 持仓情况(总价值)、持仓数、盈利持仓、亏损持仓
- 首次交易、最后交易

**持仓明细表**: 市场/方向/数量/均价/现价/成本/现值/盈亏/盈亏%

**交易历史表**: 市场/方向/类型/价格/数量/金额/时间

## 🚀 部署

### Vercel 部署
1. 下载 zip 文件
2. 访问 https://vercel.com/new
3. 上传文件夹或连接 GitHub
4. 自动部署完成

### 本地使用
直接在浏览器打开 `index.html` 即可（已配置 CORS 代理）

## 📝 测试地址

排行榜上的真实钱包地址，可直接点击查看详情。

## ⚠️ 注意事项

- 数据每2分钟自动刷新
- CORS 代理用于本地测试，生产环境建议使用自己的代理
- API 有速率限制，请勿频繁刷新
